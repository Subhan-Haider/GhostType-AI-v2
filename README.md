> *The ultimate AI-powered productivity tool for automated text entry, natural typing simulation, and advanced text restoration.*

## 🚀 Key Features

*   **⚡ Nitro Neural Engine**: High-speed typing at 400+ WPM with Zero-Overhead "Ghost Time" tracking.
*   **🔄 Resume Progress**: Never lose your place. Stopping midway generates an intelligent **Resume Banner** that picks up exactly where you left off.
*   **🤖 AI Smart Fix (Hybrid)**: Restore broken/squashed text using a 17-model relay chain. Automatically attempts model variants (like `:free`) to ensure 100% connectivity.
*   **🎯 Humanized Rhythm**: Bypasses bot detection with randomized micro-delays and simulation of natural typos.
*   **⚙️ Unified Dashboard**: Manage **OpenRouter**, **HuggingFace**, and **TextSynth** keys plus engine speed from a single cohesive interface.
*   **⏳ Focus Countdown**: Clear 3-second preparation window with active visual feedback on the button you clicked.
*   **🦾 Ghost Heuristic**: Built-in regex-based restoration fallback for offline usage.

## ✨ What's New in v2.2.9

- **Frame-Lock Resuming**: Progress tracking now works perfectly even on complex multi-frame websites.
- **Throttled Persistence**: Progress is saved every 5 characters to ensure maximum accuracy without compromising browser speed.
- **404 Resolution**: Automatic model-ID normalization prevents the common "Not Found" errors in the OpenRouter free tier.
- **New Action Bar**: Sophisticated layout with a dedicated Resume Banner for better clarity.

## ⚙️ Configuration

1.  **Speed (WPM)**: Start at **200 WPM** for natural results, or use **400+** for Nitro speed.
2.  **Accuracy (%)**: 95% is a good baseline to include realistic common typos.
3.  **Engine Speed**: Adjustable delay between characters ensures compatibility with all web forms.
4.  **API Keys**: Connect **OpenRouter** (primary), **HuggingFace**, or **TextSynth** for advanced text restoration.

## 📦 Installation

1.  **Clone or Download** this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (top right toggle).
4.  Click **Load unpacked**.
5.  Select the extension folder.

## 🛠 Usage

1.  Open the **GhostType AI** popup.
2.  **Prepare Content**: 
    - Type/Paste into the "Typing Content" field.
    - Click **📥 Collect** to grab text from the current page.
    - Click **🤖 AI Fix** to perfect your text with AI.
3.  **Start Engine**:
    - **Start New**: Resets progress and starts from the beginning.
    - **Resume Typing**: (Appears automatically) Picks up from your last character.
4.  **The Countdown**:
    - **Wait 3 seconds**: Watch the visual timer on the button.
    - **Focus Target**: Click inside the text box where you want to type.
5.  **Control**: Hit **Stop** at any time to pause. The Resume Banner will wait for you.
